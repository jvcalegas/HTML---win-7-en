Windows 7 Tutorial

For starters it is important to make it clear that every tutorial was performed on a virtual machine (Virtual Box) and we use the original windows iso.

1st step- Click on “New”.

2nd step- Select the machine name, folders where the files will be saved, operating system types, version and click “Next”.

3rd step- In this step we must select the RAM memory size to be used by O.S., you must take into account the amount of memory that O.S. needs and the memory available in the system. The best choice would be to choose the recommended size. After choosing the memory size, click “Next”.

4th step- In this select “Create a virtual hard disk now” or select an existing file if you have one. Click “Create”.

5th step- In this screen, select “VDI” and click “Next”.

6th step- Select “Dynamically allocated” and click “Next”.

7th step- In this screen, choose the disk size respecting the minimum required by the virtual O.S. and the host memory. Click on "Create".

8th step- After create your virtual machine , click “Start”.

9th step- In this step you will put the optical disk file of the O.S. that you chose for your virtual machine, in this case will be Windows7. Click on “Start”.

10th step- The O.S. will start. In this screen select “English”. If you want a different language, be free to chose what is better for you.

11th step- In this screen you must have to chose based in the choice you made in the lasts screen. Click “Next”.

12th step- Click on “Install now”.

13th step- Select “Windows7 ULTIMATE x64” or other that you prefer. Click “Next”.

14th step- After read the License Terms select “I accept the license terms” if you agreed and click “Next”.

15th step- In this screen you will choose “Custom(advanced)”. Or, if you like prefer, feel free to chose the one who will serve you better.

16th step- Select the place that you want to install your Windows. Click on “Next
